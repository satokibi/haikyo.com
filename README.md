Haikyo.com
====

「自殺サークル」, 「紀子の食卓」に出てくる
廃墟ドットコムというサイトのコピー。

## Requirement
  $ ruby -v
  ruby 2.3.1p112 (2016-04-26 revision 54768) [x86_64-linux]

## Usage
~~~$ git clone https://github.com/satokibi/haikyo.com.git~~~

~~~$ cd haikyo.com~~~

~~~$ rails s~~~
